Project Raah - Assistive Object Interaction

Open CV

Emb. Software

Raah is an assistive device designed to help visually impaired users identify and navigate toward objects in their environment. The system performs real-time object detection using OpenCV and TensorFlow-based deep learning models, enabling accurate recognition of everyday items.

Once an object is identified, Raah uses a two-stage haptic feedback algorithm to guide the user safely toward the target. The first stage provides directional cues, while the second stage offers distance-based feedback, ensuring intuitive and reliable navigation without visual input.

Built to be lightweight, affordable, and user-friendly, Raah aims to improve independence and situational awareness for visually impaired individuals.
