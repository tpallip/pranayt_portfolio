Project Raah

Object detection and guidance for visually impaired

Emb. Software

Controls

Mech

A wristband wearable with camera for object recognition. Also fitted with eight haptic motors to guide the hand of the user towards the object. The purpose is to help the visually impaired interact with the environment, especially daily-objects, in a closed-loop control

Platform
Raspberry Pi 3B, PiCAM.

Vibration motors controlled by arduino

User-validation in association with various NGOs for visually impaired

Requirements Study Object Recognition Usability Study User-validation Embedded C Python
